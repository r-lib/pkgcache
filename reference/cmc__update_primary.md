# Update the primary cache from the replica cache

Update the primary cache from the replica cache

## Usage

``` r
cmc__update_primary(self, private, rds, packages, lock)
```

## Arguments

- self:

  self

- private:

  private self

- rds:

  Whether to update the RDS file.

- packages:

  Whether to update the PACKAGES files (+ ETag files).

## Value

Nothing.
